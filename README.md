# Project Link [Fun Chat](https://rolling-scopes-school.github.io/suficks-JSFE2023Q4/fun-chat/)

## FunChat is an innovative application for communication between browser users, created using WebSockets technology. Chat provides instant and smooth communication without lag.

### Key Features: 
 - *Instant Communication*: With WebSockets, messages are delivered instantly, allowing for a smooth and continuous flow of communication between all users.
 - *Flexibility and Convenience*: FunChat allows users to write, delete, edit and add chat messages without restrictions. This gives you complete control over your communication experience.
 - *Unlimited number of users*: No matter how many users join the chat, FunChat provides stable and reliable communication for everyone.
 - *Simple Interface*: The intuitive user interface makes using WebChatX easy and enjoyable.
 - *Security and Privacy*: All messages are transmitted over a secure channel, ensuring the privacy and security of your communications.

### ❗❗❗ To view the deploy, you need to [clone the repo with the server](https://github.com/rolling-scopes-school/fun-chat-server/tree/main) and keep the server running while chating. Have fun!
